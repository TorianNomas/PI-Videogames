Aplicación que utiliza la API [Rawg.io](https://Rawg.io) para mostrar un listado de juegos.

### incluye:
- filtrado 
- búsqueda
- ordenamiento 
- creación de nuevos juegos

### Tecnologías utilizadas:

- [ ] React
- [ ] Redux
- [ ] Express
- [ ] Sequelize - PostgreSQL
